console

    ------------ funciones registro
    - assert() - Aparece un mensaje de error en la consola si la afirmación es falsa. si la afirmación 
                es verdadera, no aparecera nada. (NO ESTANDAR).
    - clear() - Limpiar la consola.
    - error() - Muestra  un mensaje de error en la consola web.
    - info() - Emite un mensaje informativo a la consola web. En Firefox y Chrome, se muestra un pequeño
                icono "i" junto a estods elementos en el registro de la consola web.
    - log() - Mustra un mensaje en la consola web (o del intérprete de JavaScript).
    - table() - Esta afuncion toma un argumento obligatorio: data, que debe ser un array o un objeto, y
                y un parámetro adicional: columns y nos muestra una tabla en la consola.
    - warn() - Imprime un mensaje de advertencia en la consola web.
    - dir() - despliega una lista interactiva de las propiedades del objeto JavaScript especificado.
            [NO ESTANDAR]


    ------------ fuciones de conteo
    - count() - regisra el numero de veces que se llama count(). esta funcion  toma como argumento  
                opcional una etiqueta.
    - countReset() - resetea el contador console count()

    ------------ fuciones de agrupación
    - group() - Crea un nuevo grupo en linea en registro de la consola web.
    - groupEnd() - Remueve un grupo en la linea en el registro de la consola web.
    - groupCollapsed() - Crea un grupo en linea pero contraido, el usuario debe expandirlo para verlo.


    ------------ fuciones de temporización
    - time() - inicia un temporiador.
    - timeEnd() - Registra el valor actual de un temporizador.
    - timeLog() - Define un temporizador.
